# JSON Data Schema

`site/data/shortages.json` is the public dashboard dataset generated from the openFDA drug shortages API. It is regenerated by the scheduled GitHub Actions refresh workflow from public FDA/openFDA data.

Machine-readable schema:

[`docs/shortages.schema.json`](shortages.schema.json)

Public source:

https://api.fda.gov/drug/shortages.json

openFDA documentation:

https://open.fda.gov/apis/drug/drugshortages/

## Top-Level Object

| Field | Type | Notes |
| --- | --- | --- |
| `schema_version` | integer | Project-owned schema version. Increment when this generated JSON shape changes incompatibly. |
| `generated_at` | string | UTC ISO 8601 timestamp for when Rx Shortage Radar generated this dataset. |
| `source` | object | Source metadata copied from openFDA plus project source URLs. |
| `summary` | object | Aggregate counts derived from `records`. |
| `records` | array | Public shortage records normalized for search and display. |

## `source`

| Field | Type | Notes |
| --- | --- | --- |
| `name` | string | Human-readable source name. |
| `api_url` | string | openFDA endpoint used by the refresh. |
| `docs_url` | string | openFDA documentation URL. |
| `terms_url` | string or null | Source terms URL from openFDA metadata. |
| `license_url` | string or null | Source license URL from openFDA metadata. |
| `last_updated` | string or null | Date reported by openFDA metadata. |
| `disclaimer` | string or null | Disclaimer reported by openFDA metadata. |

## `summary`

| Field | Type | Notes |
| --- | --- | --- |
| `total_records` | integer | Number of records in `records`. |
| `status_counts` | object | Mapping of FDA status string to count, derived from `records[].status`. |

## `records[]`

| Field | Type | Notes |
| --- | --- | --- |
| `id` | string | Project-generated stable identifier derived from public record fields. |
| `generic_name` | string | FDA/openFDA generic name for the shortage item. |
| `brand_names` | array of strings | Brand names from `openfda.brand_name`. |
| `status` | string | FDA shortage status, such as `Current`, `Resolved`, or `To Be Discontinued`. |
| `update_type` | string | FDA update type when available. |
| `update_date` | string or null | Normalized `YYYY-MM-DD` date when available. |
| `initial_posting_date` | string or null | Normalized `YYYY-MM-DD` date when available. |
| `discontinued_date` | string or null | Normalized `YYYY-MM-DD` date when available. |
| `package_ndc` | string | Package NDC for this shortage row when available. |
| `package_ndcs` | array of strings | Package NDCs from `openfda.package_ndc`. |
| `product_ndcs` | array of strings | Product NDCs from `openfda.product_ndc`. |
| `rxcuis` | array of strings | RxNorm Concept Unique Identifiers from `openfda.rxcui`. |
| `substances` | array of strings | Substance names from `openfda.substance_name`. |
| `routes` | array of strings | Routes from `openfda.route`. |
| `therapeutic_categories` | array of strings | FDA therapeutic categories from the shortage record. |
| `dosage_form` | string | Dosage form from the shortage record. |
| `presentation` | string | FDA presentation text. |
| `company_name` | string | Company name from the shortage record. |
| `related_info` | string | FDA related information text when available. |
| `source_url` | string | Source API URL. |
| `search_text` | string | Project-generated normalized text used by the static dashboard search. |

## Stability Notes

- The file is regenerated from public openFDA records, so counts, dates, and individual records can change whenever FDA updates the source feed.
- `schema_version` is controlled by this project. Consumers should check it before relying on field-level behavior.
- Empty source values are usually represented as empty strings, empty arrays, or `null`, depending on the field.
- This dataset is for public data exploration only. It is not medical advice and should not be used for medical, clinical, procurement, or patient-care decisions.
